# College Admission Management System

A console-based system developed in C++ using OOP principles. It automates college admissions, fee calculation, and department-wise student management.

## Features
- Automated roll number generation based on department & section
- Department handling (CS, Engineering, Medical) with multiple sections
- Student records including subjects, marks, percentage calculation
- Scholarship and fee calculation engine
- Menu-driven CLI interface with clean output

## Technologies Used
- C++ with Classes, Objects, Enums, Vectors
- OOP, Encapsulation, Modular Functions

## How to Run
1. Open terminal
2. Compile: `g++ main.cpp -o CollegeSystem`
3. Run: `./CollegeSystem`
